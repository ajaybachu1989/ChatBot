Cost Center

Fields and Descrptions

Cost Center:

The smallest areas of responsibilities for costs by department, function, assigned to a profit center and company code.

Field Name Field Description Data Type Field Length
Company Code The company code that represents a cost center's legal entity or balanced set of books. Company Code defines home country, fiscal year, currency, language and chart of accounts of the cost center. CHAR 4
Cost Center Name Key uniquely identifying a cost center. CHAR 10
Cost Center Id Key uniquely identifying a cost center. CHAR 10
Profit Center Key which together with the controlling area uniquely identifies a profit center. CHAR 10
Short Description General description of the cost center using standard and unique names to convey a clear purpose of the cost center. CHAR 20
Long Description The cost center description is used for online displays and evaluations that have sufficient space. CHAR 40
Person Responsible(Budget Manager) The Budget Manager person responsible for the given cost center. CHAR 20
User Responsible(Financial Anaylst) The user ID of the financial analyst user responsible for the cost center. CHAR 12
Cost Center Owner The senior leader accountable for the expense management and operations of a given cost center (director and above). CHAR 20
